### 使用技術

- 以 `Vue 3` 與 `Vue Vite` 架設開發。
- 使用 `Bootstrap 5` 作為 CSS 框架，以進行切版。
- 加入 `ESLint` 統一程式碼結構。
- 使用 `Pinia.js` 傳遞跨元件資料，作為狀態管理工具。

### 使用套件

- `Vue 3`
- `Vue Vite`
- `Vue Router`
- `Vue Axios`
- `vue-loading-overlay`
- `Vee Validation`
- `pinia`
- `Bootstrap 5`
